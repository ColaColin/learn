What is a quaternion? ijk form?
A quaternion has 4 components: $ a = (q_0, q_1, q_2, q_3) $
It is useful to represent as the scalar component and a 3d vector: $ (q_0, q) $ with $q = (q_1, q_2, q_3)$.
Similar to complex numbers it can be represented as a sum and a few rules:
$a = (q_0 + iq_1 + jq_2 + kq_3)$ mit $i^2 = j^2 = k^2 = ijk = -1$
How do you add quaternions together?
Quaternions a and b:
$a = (q_0, q)$
$b = (p_0, p)$
The sum then is: $a + b = (q_0 + p_0, q+p)$
How can you multiply quaternions?
Quaternions a and b:
$a = (q_0, q)$
$b = (p_0, p)$
$a \cdot b = (p_0 q_0 - p \cdot q, q_0 p + p_0 q + p \times q)$
What is the multiplicative identity for quaternions?
$(1, 0)$
How is the conjugate defined?
Quaternion $a = (q_0, q)$
Conjugate $a^* = (q_0, -q)$
How is the inverse of a quaternion defined?
Quaternion $a = (q_0, q)$
Inverse $a^{-1} = \frac{1}{|a|^2}(q_0, -q)$
How can quaternions be used for rotations?
A point $p = (x,y,z)$ can be rotated around an axis A, with $|A| = 1$ by angle $\alpha$ through the origin:
  1. Interpret the point as a quaternion with 0 scalar part: a = (0, p)
  2. The rotation is represented as $ r=(\frac{cos(\alpha)}{2}, \frac{sin(\alpha)}{2}A) $
  3. The rotation can be done by $a' = r a r^{-1}$
  4. The vector part of the new quaternion a' is the result
  5. To rotate around an axis that is located somewhere in space first translate the point to the origin, rotate it there and then translate it back
How can rotations be appended to each other using quaternions?
By multiplying them together.
What are the advantages of using quaternions for rotations?
  • Considerably less multiplications, thus faster computation
  • Easy interpolation between rotations, i.e. SLERP with quaternions.
  • No gimbal lock
