The Cohen Sutherland Algorithm determines a simple code for each point of a line segment that encodes where it is relative to the clipping window.
Given a point $P = (x, y)$, the out code bits $b_1 b_2 b_3 b_4$ are:
$ b1 = isNeg(y_{max} - y) $
$ b2 = isNeg(y - y_{min}) $
$ b3 = isNeg(x_{max} - x) $
$ b4 = isNeg(x - x_{min}) $
with $ isNeg(x) := \begin{cases} 0 & \text{if } x \geq 0 \\ 1 & \text{if } x \lt 0 \end{cases} $
The line segment is inside, if the code of both points is 0000,
it is outside if the logical and of the both codes is not 0000,
else it intersects the window and needs further clipping.
